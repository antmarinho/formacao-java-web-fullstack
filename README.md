# JSP (JavaServer Pages)

- O módulo **JSP (JavaServer Pages)** contém exemplos práticos para aprendizado de desenvolvimento web usando tecnologias JSP no backend Java.

## 📚 Conteúdo

O que você vai encontrar neste módulo:

- Páginas JSP de exemplo para exibir conteúdo dinâmico.  
- Integração com **servlets** para lógica de controle.  
- Uso de expressões JSP e boas práticas de separação entre lógica e apresentação.  
- **Geração de relatórios em PDF com JasperReport Studio**.  
- **Visualização de gráficos interativos com Chart.js**.  
- Exemplos de formulários, parâmetros de requisição e sessão, roteamento simples, etc.

# JSF (JavaServer Faces)

- O módulo **JSF (JavaServer Faces)** contém exemplos práticos para o aprendizado de desenvolvimento web utilizando a tecnologia JSF no backend Java, proporcionando uma base sólida para a construção de aplicações web com interface rica e interativa.

## 📚 Conteúdo

O que você vai encontrar neste módulo:

- Introdução ao JSF: Como configurar e criar o primeiro projeto JSF.
- Uso de Managed Beans: Entendendo o papel dos Managed Beans na construção de componentes e na gestão de dados da aplicação.
- Navegação de Páginas: Como gerenciar a navegação entre páginas e definir regras de roteamento com JSF.
- Formulários e Validação: Exemplos de como criar formulários interativos com JSF e realizar validação de campos.
- Integração com Banco de Dados: Conectando a aplicação JSF com um banco de dados para persistência de dados.
- Exibição Dinâmica de Dados: Utilizando componentes JSF para exibir dados de forma dinâmica nas páginas.
- Boas Práticas de Desenvolvimento: Separação entre lógica de negócios e camada de apresentação, com uso de FacesContext e Managed Beans para melhor organização do código.
- Exemplo de PrimeFaces: Integração com PrimeFaces para melhorar a interface com componentes visuais avançados.
- Este projeto é uma excelente introdução ao desenvolvimento de aplicações web utilizando o framework JSF, com exemplos práticos e código bem estruturado.
